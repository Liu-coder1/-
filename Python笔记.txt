Python不同于其它语言的语法：

1.for循环  in range后面的范围取头不取尾，例如 for i in range（1,3），i的取值为1,2。

2.“且” 表示为‘and’。 

3.python中的数字类型是不可变数据。意思说的是数字类型数据在【内存】中是不会发生改变，当变量值发生改变时，会新申请一块内存赋值为新值，然后将变量指向新的内存地址。而在C语言中，a指向的内存地址并不会发生改变，而是改变内存的内容。（Python中变量不存在自增运算符，自增方式：a = a+1）

4.Python 中，变量是以内容为基准而不是像其它语言中以变量名为基准，所以只要你的数字内容是不变的，不管你起什么名字，这个变量的 ID 是相同的（id（a）返回对象a的内存地址）

5.input() 和 raw_input() 这两个函数均能接收 字符串 ，但 raw_input() 直接读取控制台的输入（任何类型的输入它都可以接收）。而对于 input() ，它希望能够读取一个合法的 python 表达式，即你输入字符串的时候必须使用引号将它括起来，否则它会引发一个 SyntaxError 。新版Python里面已经没有raw_input了，用input。

6.chr() 用一个范围在 range（256）内的（就是0～255）整数作参数，返回一个对应的字符。

7.使用print(obj)在console上打印对象的时候，实质上调用的是sys.stdout.write(obj+'\n')，print在打印时会自动加个换行符。

8.str() 函数将对象转化为适于人阅读的形式。

9.capitalize（）将字符串首字母大写、casefold（）/lower（）字符串大写变小写、upper（）字符串全部大写

10.center（n，“-”）字符串居中显示宽度为n，用-填充

11.count（"s"）计算某字符串中s出现几次，可以是多个字符；可加范围控制

12.endswith（""）判断字符串以什么结尾，返回True或者False

13. isalpha() 方法检测字符串是否只由字母组成。
    isspace() 方法检测字符串是否只由空格组成。
    isdigit() 方法检测字符串是否只由数字组成。

14.reduce（）求和函数，需要import functools。

15.lambda x,y : x + y，lambda表达式：创建一个函数带参xy，返回x+y。
